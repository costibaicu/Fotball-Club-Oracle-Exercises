# Fotball Club Oracle Exercises

Oracle SQL & PL/SQL project for the design and implementation of a database management system for a football club (FCSB).

## About

Academic project developed for the Database Management Systems (SGBD) course. It covers the full lifecycle of a relational database — from design and table creation to complex queries and PL/SQL automation.

## Database Schema

The database models a football club management system with the following tables:

- **MEMBRU** — club members
- **SECTIE** — club sections (seniors, juniors, medical)
- **ANTRENOR** — coaches
- **SALA** — training facilities
- **ABONAMENT** — membership plans
- **GRUPA** — training groups
- **PLATA** — payments
- **ACTIVITATE** — training activities
- **MEMBRU_ABONAMENT** — member subscriptions
- **MEMBRU_GRUPA** — member group assignments
